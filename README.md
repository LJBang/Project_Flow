# make_RESTAPI

JAVA Spring Boot를 활용하여 REST API 구축을 공부하기 위한 프로젝트

# Tech
JAVA  
SpringBoot maven  
postgresql  

## Env
`JDK 11.0.10`  
`Postgresql 11.12`  
`postgresql JDBC 42.2.23`  
`Spring Boot 2.5.2`  
- Spring Boot dependency
    - Spring Boot DevTools  
    - Lombok  
    - Spring Web  
    - Spring Data JPA  
    - PostgreSQL Driver
